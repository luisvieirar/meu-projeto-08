# meu-projeto-08

 Resumo do Projeto
Desenvolvi um sistema de cadastro de funcionários completo com PHP e PostgreSQL.

O que o sistema faz:
🔐 Login de usuário (admin/123456)

➕ Cadastrar novos funcionários

✏️ Editar dados existentes

🗑️ Excluir funcionários

🔍 Buscar por nome

📧 Enviar e-mail direto da listagem

Telas criadas:
Login - entrada do sistema

Cadastro - formulário para adicionar/editar

Listagem - tabela com todos os funcionários

Dificuldades resolvidas:
O cadastro não salvava → implementei INSERT/UPDATE

O login não funcionava → criei um arquivo separado para processar

Botões sem ação → adicionei funções onclick

Busca não filtrava → usei ILIKE no SQL

Tecnologias:
PHP

PostgreSQL

HTML/CSS

JavaScript básico
